## Build
 ```sh
  gcc -W -Wall -Wextra -pedantic -o cing-cong cing-cong.c -Iinclude -lpthread -Llib -lSDL2 -lSDL2main
   ```
## Movements
playerRight moves with keyUp and keyDown, and playerLeft moves with w and s
